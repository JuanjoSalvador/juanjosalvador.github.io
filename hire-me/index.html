A bit of background

I started coding when I was 8 and I found out several BASIC games on Public Library's books, which quickly generated an interest on me. At 12 I created and published my very first website at Geocites, and started reading about PHP, but never went too far.

My philosophy

KISS. Keep It Stupidly Simple. Because life is already too complicated, just make it simplier.

I always try to apply the zen of Python, since this is a good way to keep your code clean, scalable and easier to maintain.

Tech I love

There are several technologies, frameworks and tools I love, and I would like to keep working with them.

  • Languages: Python, Java, JavaScript, HTML, CSS
  • Frameworks: Django, Django REST Framework, Bulma, Vue
  • Tools and platforms: Git, GitHub, Docker, Linux

Thoughts about AI

While I consider AI a huge advance in human technology, I'm concerned and aware about its dangers and ethics using it. I will not active tell anyone to not use it, but overall I don't want to turn it into a basic tool in my life, specially speaking about code generation.

Remote-friendly

I am specially open to remote-friendly positions, since I live in a small city and moving from here at this moment is a big problem.
